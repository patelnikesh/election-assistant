# 🗳️ Election Assistant

Election Assistant is a modern, interactive web application designed to empower citizens with comprehensive information about the election process. Built with a focus on user experience, it simplifies complex civic information into engaging, easy-to-digest components.

![Election Assistant Mockup](https://images.unsplash.com/photo-1540910419892-4a36d2c3266c?auto=format&fit=crop&q=80&w=1200&h=400)

## ✨ Features

- **🤖 AI Chat Assistant**: An intelligent interface to answer your specific questions about voting and election procedures.
- **📚 Educational Guide**: A structured walkthrough of the voting process, from registration to casting your ballot.
- **⏳ Interactive Timeline**: Visualize key election dates and deadlines to ensure you never miss an important milestone.
- **🧠 Knowledge Quiz**: Test your election knowledge with an interactive quiz designed to educate and engage.
- **🃏 Flashcards**: Quickly learn important election terms and concepts with our dynamic flashcard system.
- **📱 Responsive Design**: Fully optimized for desktops, tablets, and smartphones.

## 🚀 Tech Stack

- **Frontend**: React.js with Vite for blazing-fast development and performance.
- **Styling**: Modern CSS with a focus on premium aesthetics, featuring smooth transitions and a clean UI.
- **Icons**: [Lucide React](https://lucide.dev/) for consistent and beautiful iconography.
- **Deployment**: Ready for Firebase Hosting and Google Cloud Run.

## 🛠️ Installation & Setup

To run the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/patelnikesh/election-assistant.git
   cd election-assistant
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🐳 Docker Deployment

The project includes a `Dockerfile` for easy containerization:

```bash
docker build -t election-assistant .
docker run -p 8080:8080 election-assistant
```
